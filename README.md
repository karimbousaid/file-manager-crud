# File Manager CRUD

## Projektübersicht

Webanwendung zur Datei Verwaltung mit:

- JWT Authentifizierung
- Datei Upload mit Validierung
- SQLite als Datastore
- FastAPI Backend
- Vue 3 Frontend
- OpenAPI (Swagger)
- Containerisierung mit Docker
